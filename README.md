# VigiVerse

VigiVerse is an AI-enabled pharmacovigilance platform that allows the general public to report adverse drug reactions (ADRs).

## Getting Started

Since this project was created manually (due to missing Node.js/npm in the environment), you will need to install the dependencies first.

### Prerequisites

- Node.js 18.17 or later
- npm (comes with Node.js)

### Installation

1.  Open a terminal in this directory.
2.  Run `npm install` to install all dependencies.

### Running the App

1.  Run `npm run dev` to start the development server.
2.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Landing Page**: Modern introduction to the platform.
- **Dashboard**: User overview of reports and rewards.
- **ADR Reporting**: Structured form with AI verification simulation.
- **Gamification**: Leaderboard and points system.

## Tech Stack

- Next.js 14 (App Router)
- TailwindCSS
- Lucide React
- TypeScript
